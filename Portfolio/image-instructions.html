Image Setup Instructions

Download the bear coding image

I wasn't able to directly save the image file through our interface. Please follow these steps to add the image to your portfolio:

  1. Download the bear image from the original source or save it from our conversation
  2. Rename it to bear-coding.jpg
  3. Place it in the following directory in your project:
    C:\Users\Abernaughty\Documents\GitHub\git-maber\Portfolio\static\images\

Alternative: Use a placeholder image temporarily

If you'd like to see how the site works before adding the specific image, you can modify the CSS to use a placeholder service:

.hero-background {
position: absolute;
top: 0;
left: 0;
width: 100%;
height: 100%;
background-image: url('https://placebear.com/1200/800');
background-size: cover;
background-position: center;
background-repeat: no-repeat;
filter: brightness(0.7);
z-index: -1;
}

Just replace the url part in your CSS with the placeholder above. This will display a random bear image temporarily.

Once you've added the image, restart your development server (if it's running) with npm run dev to see the changes.
